- Booksfy – Sistema de Gerenciamento de Biblioteca Pessoal

O Booksfy é um sistema web moderno para organização de leitura, permitindo cadastrar livros, acompanhar progresso, registrar avaliações, aplicar filtros avançados e visualizar estatísticas.
Projeto desenvolvido como parte do A3 de Engenharia de Software, com documentação completa de requisitos, escopo e métricas de qualidade.

- Funcionalidades Principais

Cadastro e autenticação de usuários

Cadastro, edição e exclusão de livros

Status: Quero Ler, Lendo, Lido

Avaliação por estrelas e comentários

Progresso por capítulos

Categorias personalizadas

Busca e filtros avançados

Dashboard com gráficos e estatísticas

Interface responsiva

- Tecnologias

React + TypeScript + Vite

Tailwind CSS

shadcn/ui + Radix UI

Supabase (Auth + PostgreSQL + RLS)

TanStack Query

Recharts
